## 🚀 How to Run

### Quick Start (3 Terminals)

**Terminal 1 - MongoDB:**
```bash
mongod
```

**Terminal 2 - Python Backend:**
```bash
cd backend
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

**Terminal 3 - Node Frontend:**
```bash
cd frontend
npm start
```

**Access:** http://localhost:3000
